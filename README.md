# Monkey Interpreter (Rust Implementation)

This repository contains a **Rust** implementation of the [Monkey programming language](https://monkeylang.org/) interpreter. It is a project that focuses on the core components of language processing, including lexical analysis and parsing.


## Language Features

- **Data Types:** Supports integers, booleans, strings, arrays, and hash maps.
- **Variable Bindings:** Uses let statements to bind values to identifiers.
- **Functions:** Includes first-class and higher-order functions, built-in functions, and support for recursion.
- **Advanced Logic:** Features closures, conditionals, and both implicit and explicit return statements.

### Added
- Mutable variables
## Usage

**Run the interactive interpreter**
```bash
$ cargo run
```
**Run tests suite**
```bash
$ cargo test
```
